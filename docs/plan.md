# Project Plan: Book Tracker

## 1. Project Goal

Create a web application for tracking books that allows users to:  

- Add books they are reading or planning to read  
- Track reading progress and ratings  
- Categorize books by genres or tags  
- Search and filter books  
- View their personal library  

---

## 2. Functional Requirements

### 2.1 Users
- Register / Login / Logout / Delete Profile
- Each user sees only their own books  
- Edit user profile (password)

### 2.2 Books
- Add a new book: title, author, genre, total pages, rating  
- Edit and delete books  
- Track reading progress (e.g., % read)  

### 2.3 Filters and Search
- Search by title and author  
- Filter by genre, tags or rating  

### 2.4 Additional Features
- Sort by date added, rating, or progress  
- Favorite books / bookmarks  
- Comments or notes for books 
- Front-end and back-end form validation  

---

## 3. Non-Functional Requirements

- **Frontend:** responsive design, user-friendly interface  
- **Backend:** Node.js + Express, REST API  
- **Database:** PostgreSQL 
- **Authentication:** bcrypt for passwords, sessions 
- **Error Handling:** validate inputs, proper server responses  
- **Deployment:** Render and Supabase

---

## 4. Technologies

- **Backend:** Node.js, Express  
- **Frontend:** React, SCSS, fetch API, Reactbits
- **Database:** PostgreSQL
- **Authentication:** bcrypt + sessions 
- **Deployment:** Render, Supabase
